== Description ==
[Scrollbar Customizer](https://wpmighty.com/wordpress-plugins/scrollbar-customizer) is an advanced WordPress plugin that let you easily customize and change the design and control of the scrollbar in your website.

## The Key Features of Scrollbar Customizer ##
⭐ Customize dragger height
⭐ Customize bar width and side space
⭐ Customize bar background color
⭐ Customize dragger color
⭐ Change bar border radius
⭐ Compatible with all page builders
⭐ Compatible with all Wordpress themes
⭐ Gutenberg compatible
⭐ Include pot file for translation
⭐ Compatible with last Wordpress version
⭐ Lightweight and fast with no jQuery
⭐ Compatible with all modern browsers


## Pro version Features ##
⭐ Dark mode theme support
⭐ Gradient dragger support
⭐ Change wheel speed
⭐ Size Animation on hover
